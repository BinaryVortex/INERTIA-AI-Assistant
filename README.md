# INERTIA AI Assistant

A voice-controlled AI assistant built using HTML, CSS, and JavaScript. INERTIA can perform various tasks through voice commands, including opening websites, searching the web, providing time and date, and more.

## Features

- **Voice Recognition**: Uses the Web Speech API to recognize voice commands.
- **Text-to-Speech**: Responds audibly using speech synthesis.
- **Time-Based Greetings**: Greets users based on the current time of day.
- **Web Integration**: Opens popular websites like Google and Instagram.
- **Search Functionality**: Searches Google or Wikipedia for queries.
- **Utility Commands**: Provides current time, date, and opens the calculator.
- **Fallback Search**: Falls back to Google search for unrecognized commands.

## How to Use

1. Open `index.html` in a modern web browser that supports the Web Speech API (e.g., Chrome, Edge).
2. Click the "Talk" button or allow microphone access.
3. Speak your command clearly.
4. INERTIA will respond via voice and/or open relevant pages.

## Supported Commands

- "Hey" or "Hello": Greets the user.
- "How are you": Responds with a friendly message.
- "Name": Introduces itself as Inertia.
- "Open Google": Opens Google in a new tab.
- "Open Instagram": Opens Instagram in a new tab.
- "What is/Who is/What are [query]": Searches Google for the query.
- "Wikipedia [topic]": Opens Wikipedia page for the topic.
- "Time": Speaks the current time.
- "Date": Speaks the current date.
- "Calculator": Opens the system calculator.
